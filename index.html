PLE Project year 3 (Work in Progress)

Currently I'm working a project for school where I'm going to create a short virtual experience where you play as a bartender and engage in interactions based on how you judge the appearance of avatars. Your task is to serve drinks to avatars based on specific prompts, like "Serve a drink to someone who looks friendly", While you collect points for doing your tasks correctly. At the end of the experience, you’ll get insight into your decisions and how your interactions compare to those of other players.That’s how I want to gain insight into biases and perception in virtual environments.

I've been making this game in Unity using C#, which has been so much fun. I've worked in Unity before, but I never coded my own features until now, and I've really been enjoying it. I was really nervous when I started but after trying out some things I got a lot more comfortable in the work flow of coding in Unity, which is something I'm actually a little proud of. I still need a little help from Copilot, but I've already learned so much while working on this game over the past few weeks! I showed myself I can create things I never thought I would ever be able to, which is the coolest thing ever to me! I can't wait to see what the final result will look like. I'm still working on this project, but below you can see what I have so far.

This is the first prototype where I tried to get an interaction of picking up a glass and then getting visual feedback to work!

This is a bit of progress from a couple of weeks after the first prototype. I added features like a prompt generator, bar trigger (to trigger the prompt generator), a timer, and a score counter. During this time, I was really struggling to get things working in my project. Getting comfortable with Unity took a little while, so this prototype looks far from perfect—but it still did the job. I was able to test the features I needed using the XR Simulator in Unity, which lets you test your game without a VR headset (which I didn’t always have access to).

The third prototype and one I'm actually really proud of!! I did the video in dutch for my studies but here's a little translation of what is happening in the video: When you approach the bar, a prompt appears, which I can add myself very easy through a prompt manager. You can pick up a glass and give it to one of the avatars. While holding the glass, highlight effects appear—these are linked to avatar tags (for now these are the same avatars but they already have different tags). The system records which avatar you gave the glass to and which prompt was active at the time. You can see that in the console log in the video! You can also see me using a painterly shader in this prototype—an oil paint shader I added to make the game look a bit nicer, and it really did! But sadly, it was too demanding for deployment on the Oculus Quest 2/3, so I had to remove it... Anyways once you put the glass down, everything resets: the old prompt is cleared, and a new one is generated the next time you come close to the bar. :D
